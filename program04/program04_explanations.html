Bootstrap Explanations

The Button Groups Category

Classes Used: btn-group

The btn-group class is used to display buttons side-by-side, instead of each button on their own line. Grouping the buttons together allows anyone who visits the site to intuitively interact with them.

The Utility: Positioning Category

Classes Used: float-left

The float-left class is used to left-align the navigation bar brand element on all viewport sizes. This ensures the picture stays on the left side of the screen on any display a user visits the website on.

The Navbar Category

Classes Used: navbar-brand, collapse navbar-collapse, navbar-toggle

The navbar-brand class allows customization of the navigation bar icon to promote a given brand, for which I have used a book image. The navbar-toggle class enables the navigation bar to toggle into a "hamburger" icon on smaller screens, while the collapse navbar-collapse class sets up the functionality which expands and collapses the navigation bar list items for smaller viewport sizes.

The Grid Category

Classes Used: container, container-fluid, row, col-md-6

The container class gives the webpage its overall structure. The container-fluid class does the same, but uses the full width of the screen. The row class splits a given container into horizontal sections, while the col-md-6 class splits a given container into two vertical sections. The container classes allow content to resize based on the current display resolution.

Bootstrap vs WordPress

Bootstrap is a framework which allows more streamlined web development. It is more flexible and configurable than WordPress. Bootstrap allows straightforward customization for any given webpage. The downside of Bootstrap is that it requires more technical knowledge and thus is not an ideal choice to build a website and hand off to someone else to maintain... you will likely get calls when something breaks and they don't know how to fix it.

WordPress, on the other hand, is WYSIWYG Editor. It allows non-technical users to create professional-looking websites that are easy to maintain. There are still a lot of configuration options, but there aren't as many minor tweaks available compared to Bootstrap. WordPress allows website editing and hosting, while Bootstrap only provides the framework to program websites.
