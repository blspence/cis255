The container Class

Used to break up the webpage into sections.

The container-fluid Class

Used to break up the webpage into sections that format based on the screen size.

The col-md-6 Class

Used to split horizontally.

The row Class

Used to split vertically.

The img-responsive Class

Used for responsive images.

The navbar navbar-inverse Class

Used to configure the main navigation bar.

The navbar-toggle Class

Used to toggle navigation bar list items.

The navbar-brand Class

Used to define brand in navigation bar.

The collapse navbar-collapse Class

Used to define navigation bar list items.

The nav navbar-nav navbar-right Class

Used to right-align navigation bar elements.

Bootstrap vs WordPress

Bootstrap is a framework which allows more streamlined web development. It is more flexible and configurable than WordPress. Bootstrap allows complete customization for any given webpage. The downside is that it requires more technical knowledge and thus is not an ideal choice to build a website and hand off to someone else to maintain... you will likely get calls when something breaks and they don't know how to fix it.

WordPress, on the other hand, is reliable and still configurable. There aren't as many minor tweaks, but it allows non-technical users to create professional looking websites that are easy to maintain. WordPress allows website editing and hosting, while Bootstrap is only providing the framework.
