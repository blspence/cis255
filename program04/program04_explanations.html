Bootstrap Explanations

The Button Groups Category

Classes Used: btn-group

This category was used to display buttons side-by-side, instead of each button on their own line. Grouping the buttons together allows anyone who visits the site to intuitively interact with them.

The Utility: Positioning Category

Classes Used: float-left

This category was used to left-align the navigation bar brand element on all viewport sizes. This ensures the picture stays on the left side of the screen on any display a user visits the website on.

The Navs Category

Classes Used: navbar, navbar-inverse, navbar-toggle, navbar-brand, navbar-collapse, nav, navbar-nav, navbar-right

This category presents a sort of "table of contents" to the user, which outlines the website contents and allows quick navigation to the sections mentioned in the navigation bar. This category also assists in configuring the navigation bar for different viewport sizes; the navbar will collapse into a "hamburger" icon on smaller screens.

The Grid Category

Classes Used: container, container-fluid, col-md-6, row

This category gave the webpage its overall structure. The container classes assist in providing structure within which rows and columns are stored, while allowing the content to resize based on the current display resolution.

The Collapse Category

Classes Used: collapse

This category was used to enable the navigation bar collapse functionality.

Bootstrap vs WordPress

Bootstrap is a framework which allows more streamlined web development. It is more flexible and configurable than WordPress. Bootstrap allows straightforward customization for any given webpage. The downside of Bootstrap is that it requires more technical knowledge and thus is not an ideal choice to build a website and hand off to someone else to maintain... you will likely get calls when something breaks and they don't know how to fix it.

WordPress, on the other hand, is WYSIWYG Editor. It allows non-technical users to create professional-looking websites that are easy to maintain. There are still a lot of configuration options, but there aren't as many minor tweaks available compared to Bootstrap. WordPress allows website editing and hosting, while Bootstrap only provides the framework to program websites.
