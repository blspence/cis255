Bootstrap vs WordPress

Bootstrap is a framework which allows more streamlined web development. It is more flexible and configurable than WordPress. Bootstrap allows straightforward customization for any given webpage. The downside of Bootstrap is that it requires more technical knowledge and thus is not an ideal choice to build a website and hand off to someone else to maintain... you will likely get calls when something breaks and they don't know how to fix it.

WordPress, on the other hand, is WYSIWYG Editor. It allows non-technical users to create professional-looking websites that are easy to maintain. There are still a lot of configuration options, but there aren't as many minor tweaks available compared to Bootstrap. WordPress allows website editing and hosting, while Bootstrap only provides the framework to program websites.
