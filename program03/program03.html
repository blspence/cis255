Twenty Useful CSS Properties

Brionna Spencer

The cursor Property

Discussion. This property sets the mouse pointer upon hovering over a given element. This is useful to inform the user of different functionality, such as zooming in on an image.

Demonstration.

Hover over this text to change the cursor.

The box-shadow Property

Discussion. This property adds one or more shadows to an element. It is useful for customizing element appearances.

Demonstration. A gray shadow is added to the rectangle.

The transform Property

Discussion. This property permits image transformation which aids in more expressive webpages.

Demonstration. The transform property rotates the rectangle 5 degrees.

The filter Property

Discussion. This property defines visual effects on an image. These effects allow customization which can more align with the developer's intended theme, e.g. grayscale for images which could have been taken before the creation of color film.

Demonstration.

The opacity Property

Discussion. This property modifies transparency of an element, which is useful for setting unique webpage backgrounds.

Demonstration. The opacity of the rectangle is set to .5, which is equivalent to 50% transparency.
